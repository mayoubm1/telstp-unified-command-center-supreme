# Technology Stack

## Programming Languages
- **TypeScript**: Primary language for frontend development with strict type checking
- **JavaScript**: Supporting language for configuration and legacy components
- **Python**: Backend services and data processing
- **HTML/CSS**: Web interface markup and styling

## Frontend Framework
- **React**: v19.1.0 - Core UI library
- **React Native**: v0.81.5 - Mobile development framework
- **Expo**: ~54.0.19 - Cross-platform development platform
- **React DOM**: v19.1.0 - Web rendering

## Backend Technologies
- **Supabase**: Real-time database and authentication
- **Python**: Backend service development
- **Firebase**: Additional cloud services and deployment

## Build Tools and Development
- **Vite**: Modern build tool and development server
- **TypeScript Compiler**: v5.9.2 - Type checking and compilation
- **ESLint**: v9.25.0 - Code linting and quality
- **Expo CLI**: Development and deployment tooling

## Key Dependencies
- **@supabase/supabase-js**: v2.76.1 - Supabase client library
- **@react-navigation**: v7.x - Navigation framework
- **expo-router**: v6.0.13 - File-based routing
- **react-native-reanimated**: v4.1.1 - Animations
- **react-native-gesture-handler**: v2.28.0 - Touch handling

## Development Commands
```bash
# Start development server
npm start
expo start

# Platform-specific development
npm run android    # Android development
npm run ios        # iOS development  
npm run web        # Web development

# Code quality
npm run lint       # Run ESLint

# Project management
npm run reset-project  # Reset project state
```

## Configuration Files
- **tsconfig.json**: TypeScript compiler settings with strict mode
- **package.json**: Dependencies and npm scripts
- **app.json**: Expo application configuration
- **vite.config.js/ts**: Build tool configuration
- **firebase.json**: Firebase deployment settings

## Development Environment
- **Node.js**: Required for package management and build tools
- **Expo CLI**: For mobile development and deployment
- **Python**: For backend services
- **Git**: Version control system
